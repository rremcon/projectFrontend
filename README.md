# projectFrontend

Dit is de FrontEnd software, overige applicaties en bestanden zijn terug te vinden in de .ZIP-folder.

De .ZIP-folder bevat de volgende onderdelen:

- Een Installatiehandleiding: Raadpleeg deze om te starten.
- Een FunctioneelTechnisch Ontwerp: Hierin worden ontwerpen, eisen, use cases, sequentiediagrammen en een klassendiagram weergegeven.
- Broncode FrontEnd: (dit) projectFrontend.
- Broncode BackEnd: projectBackend.
- Verantwoordingsdocument: Een omschrijving van technische keuzes, limitaties en verbetermogelijkheden.
